Text Parallax

Parallax effect build with jquery.stellar plugin.

You can easily change a parallax text effect. Just change data-stellar-background-ratio speed and text background image url

HTML:

Wow animation

Animation build with WOW.js plugin and Animate.css.

To add a new animation on scroll just set jquery class wow  and choose animate class from animate.css. Animation is fadeInRight from animate.css

Project Gallery

For gallery used Magnific Popup

How to change pop up image?

Just change image in src, change large image in href attribute. It will open in the lightbox. Don't forget to change title of project
Images are in a portfolio directory.

EXAMPLE:

You can use light overlay, for light project thumbnail

Just add class project-light and project thumbnail image will be lighter. For dark overlay - just remove class project-light
